## Flight Detail Viewing App

A simple and clean flight management application built as part of a machine test using React and Bootstrap.

---

## Features

Add new flights with validation
Filter flights:
  - Name
  - Price range
  - Departure & Arrival time
Sort flights:
  - Price
  - Duration
  - Arrival
  -Departure
- Reset filters
- Responsive and clean UI

---

##  Tech Stack

- React (Vite)
- React Bootstrap
- React Router
- Context API (State Management)

---

##  Project Structure

- `components/` → UI components (Navbar etc.)
- `pages/` → Main pages (Create, List)
- `contexts/` → Flight state management

---

##  Run Locally

npm install
npm run dev
